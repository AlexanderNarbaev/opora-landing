# Что нужно проекту «Опора»

Проект прошёл трёхлетнюю проработку. Документация готова. Архитектура спроектирована. Ниже — не абстрактный запрос, а конкретные позиции, где нужна помощь.

---

## Ключевые вакансии

### GR-директор — критическая позиция №1

Опыт взаимодействия с ФОИВами, понимание законодательного процесса, выходы на Аппарат Правительства, Минцифры, Минпромторг. Задача: провести поправки в НК РФ (ст. 149), согласовать ЭПР, обеспечить межведомственную координацию.

### CFO — высокий приоритет

Управление бюджетом 10+ млрд руб. на горизонте 5 лет. Опыт работы с институтами развития (Сколково, ВЭБ.РФ, Корпорация МСП). Due diligence, фандрайзинг.

### CISO — высокий приоритет

Аттестация ФСТЭК (класс К2, приказ №117). ГОСТ-криптография (КриптоПро CSP 5.0). 152-ФЗ (персональные данные). Пентест, сертификация госсистем.

### AI Lead — высокий приоритет

Стек: CatBoost, YandexGPT, RuBERT, Temporal Fusion Transformer, XGBoost. Управление ML-командой 3–5 человек.

## Инфраструктурные потребности

### Государственные API

- **ГИСП** — верификация промышленной продукции (ПП №719). Критичен для Витрины
- **«Честный знак»** — прослеживаемость товаров. Критичен для Цепочек
- **ОФД** — подтверждение транзакций. Ключевой элемент программы лояльности
- **Платформа цифрового рубля** — sandbox ЦБ для тестирования смарт-контрактов

### Облачная инфраструктура

Yandex Cloud / VK Cloud. Аренда GPU (16–32) для AI-модулей.

## Партнёрства

### Пилотный регион
Москва + Зеленоград. Ищем административную поддержку правительства Москвы.

### Банк-партнёр
Для спецсчёта лояльности. Опыт работы с маркированными платежами ЦБ.

### Маркетплейсы
Интеграция грейдирования через API с Wildberries и Ozon. Верифицированный бренд получает бейдж на маркетплейсе.

### Университеты
R&D-партнёрства для ML-исследований и воронка кадров.
