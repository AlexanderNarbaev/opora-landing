Платформа «Опора» — 20 сервисов (13+7)

Версия 1.0 | Июль 2026

Обзор архитектуры

Платформа «Опора» состоит из 20 сервисов, сгруппированных в 4 контура: 3 бизнес-контура и 1 инфраструктурный.

Контур Модули Назначение
Коммерция Витрина, Аналитика, Песочница, Лояльность, Платежи B2C-взаимодействие, лояльность, данные, платежи
Кооперация Кооперация, Креаторы, Кадры, Регионы B2B-мэтчинг, HR, география
Финансы и право Финансы, Право, Цепочки, Экспорт Финансы, юриспруденция, логистика, экспорт
Инфраструктура Auth, Gateway, Mock, LK, BFF, Admin, AI Техническая инфраструктура, ИИ-сервисы

Бизнес-модули

1. Опора.Витрина (catalog-service)

Параметр Описание
Назначение Каталог верифицированных российских брендов с грейдированием и кешбэком
Статус ✅ IMPLEMENTED (37 Java-файлов)

Ключевые функции: - Каталог брендов с фильтрацией по ОКПД2, ОКВЭД, региону, степени локализации - Трёхуровневое грейдирование: Золото / Серебро / Бронза - Кешбэк до 7% (3% базовый + 4% за активность) - Социальный трек (отслеживание impact-покупок) - Интеграция с «Честным знаком» и ГИСП - Мультиязычность (RU, EN, ZH)

Целевые пользователи: Граждане (B2C), представители брендов

API-эндпоинты: - GET /api/v1/brands — список брендов с фильтрацией - GET /api/v1/brands/{id} — карточка бренда - GET /api/v1/products — каталог товаров - POST /api/v1/brands/{id}/verify — запуск верификации - GET /api/v1/grades/{brandId} — грейд бренда

2. Опора.Кооперация (cooperation-service)

Параметр Описание
Назначение AI-подбор партнёров, артельные кластеры, B2B-тендеры
Статус ✅ IMPLEMENTED (100 Java-файлов)

Ключевые функции: - AI-подбор партнёров по 50+ параметрам (CatBoost) - Карточки мощностей с верификацией через Госуслуги - Артельные кластеры (объединение МСП под крупные заказы) - B2B-тендеры с AI-ранжированием - Репутационный реестр (7-балльная шкала) - Смарт-контракты для фиксации обязательств

Целевые пользователи: МСП-производители, крупные заказчики

API-эндпоинты: - POST /api/v1/matching — AI-подбор партнёров - GET /api/v1/capacities — реестр мощностей - POST /api/v1/tenders — создание тендера - POST /api/v1/clusters — создание артельного кластера - GET /api/v1/reputation/{businessId} — репутационный профиль

3. Опора.Креаторы (creators-service)

Параметр Описание
Назначение Маркетплейс для самозанятых: дизайнеры, копирайтеры, маркетологи
Статус ✅ IMPLEMENTED (51 Java-файл)

Ключевые функции: - Портфолио креаторов с верификацией компетенций - Биржа заказов с AI-мэтчингом - 0% комиссия для самозанятых, 7% для агентств - Интеграция с «Мой налог» (авто-отчётность) - Система рейтингов и отзывов - Безопасная сделка (эскроу)

Целевые пользователи: Самозанятые (16,4 млн в РФ), МСП-заказчики

API-эндпоинты: - GET /api/v1/creators — каталог креаторов - POST /api/v1/orders — размещение заказа - POST /api/v1/orders/{id}/accept — принятие заказа - GET /api/v1/creators/{id}/portfolio — портфолио - POST /api/v1/reviews — создание отзыва

4. Опора.Аналитика (analytics-service)

Параметр Описание
Назначение CRM/CDP, прогноз спроса, монетизация обезличенных данных
Статус ✅ IMPLEMENTED (32 Java-файла)

Ключевые функции: - Встроенная CRM для брендов - CDP (Customer Data Platform) — портрет потребителя - Прогноз спроса (Temporal Fusion Transformer) - Монетизация обезличенных данных (с согласия граждан) - BI-дашборды и отчёты - Сегментация аудитории

Целевые пользователи: Представители брендов, аналитики, маркетологи

API-эндпоинты: - GET /api/v1/analytics/dashboard — BI-дашборд - GET /api/v1/analytics/segments — сегменты аудитории - POST /api/v1/analytics/forecast — прогноз спроса - GET /api/v1/analytics/consumer-profile — портрет потребителя - POST /api/v1/data-monetization/consent — управление согласиями

5. Опора.Право (legal-service)

Параметр Описание
Назначение Арбитраж, AI-юрист, патентный маркетплейс, типовые смарт-контракты
Статус ✅ IMPLEMENTED (37 Java-файлов)

Ключевые функции: - AI-юрист с авто-претензиями (RuBERT) - Коллективные иски - Три уровня эскалации (медиация → арбитраж → суд) - Патентный маркетплейс - Типовые смарт-контракты с NDA - Интеграция с «Право.ru» и ФАС

Целевые пользователи: МСП, юристы, патентные поверенные

API-эндпоинты: - POST /api/v1/disputes — создание спора - POST /api/v1/legal/ai-consult — AI-консультация - GET /api/v1/patents — патентный маркетплейс - POST /api/v1/contracts/template — типовой контракт - POST /api/v1/disputes/{id}/escalate — эскалация
