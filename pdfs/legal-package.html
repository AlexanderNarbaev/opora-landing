ОПОРА — Юридический пакет Июль 2026

Юридическая рамка проекта «Опора»

Юридическая рамка проекта «Опора» обеспечивает правовую основу для деятельности платформы как национальной цифровой экосистемы с миноритарным госучастием. Пакет охватывает корпоративное управление, налогообложение, информационную безопасность и интеграцию с государственными системами.

2. Учредительные документы

Устав АО «НП «Опора»

Определяет правовой статус, структуру управления, права акционеров. Включает положения о миноритарном госучастии и «золотой акции».

Корпоративный договор

Регулирует отношения между частными и государственным соучредителями: порядок принятия решений, выход из состава, механизмы разрешения споров.

Положение о Наблюдательном совете

Состав (представители государства, частных инвесторов, независимые директора), компетенция, порядок созыва и принятия решений.

Положение о Стратегическом комитете

Консультативный орган с участием представителей Минэка, Минцифры, ЦБ РФ, отраслевых ассоциаций. Выработка стратегических рекомендаций.

Трудовые соглашения с KPI

Типовые контракты для ключевых сотрудников с привязкой вознаграждения к измеримым показателям эффективности.

3. Налоговое регулирование

Поправки в ст. 149 НК РФ

Ключевой элемент юридической рамки — освобождение операций с баллами лояльности от НДС. Это позволяет:

  • Начислять кешбэк до 7% без увеличения налоговой нагрузки на платформу
  • Не учитывать баллы как объект налогообложения при их начислении и использовании
  • Создать прецедент для программ лояльности с балльной системой как инструмента стимулирования спроса

Налоговый режим самозанятых

Модуль «Песочница» интегрирован с системой «Мой налог» (ФНС). Для самозанятых — автоматический налоговый учёт через API ФНС. Комиссия платформы — 0%.

4. Регуляторная среда

Постановление Правительства об ЭПР (2027–2032)

Экспериментальный правовой режим (ЭПР) в сфере цифровых инноваций позволяет:

  • Тестировать балльную систему лояльности без полного соответствия текущему законодательству о платёжных системах
  • Использовать маркированные платежи ЦБ РФ в пилотном режиме
  • Интегрировать цифровой рубль на ранней стадии
  • Применять смарт-контракты для B2B-сделок в рамках ЭПР

Нормативные акты ЦБ РФ

  • Маркированные платежи — специальный код транзакции для идентификации платежей на платформе «Опора»
  • Спецсчёт лояльности — отдельный счёт для учёта баллов, не являющихся денежными средствами
  • Интеграция с СБП — приём платежей через Систему быстрых платежей Банка России

9. Комплаенс и GR-стратегия

Для обеспечения юридической устойчивости предусмотрены:

  • Выделенный бюджет на GR — работа с профильными комитетами Госдумы, Минэком, Минцифры, ЦБ РФ
  • Участие в рабочих группах — по цифровой экономике, импортозамещению, поддержке МСП
  • Юридическое сопровождение — штатный юрист (Волна 1) + внешние консультанты
  • Комплаенс-мониторинг — отслеживание изменений в 152-ФЗ, 149-ФЗ, нормативных актах ЦБ РФ и ФСТЭК
  • Прозрачность — публичная отчётность о структуре собственности, финансах и KPI
